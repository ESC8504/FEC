## Getting Started

1. **Clone the repo**  
   `git clone https://github.com/Omega-outlet/FEC.git`

2. **Navigate to project directory**  
   `cd your-project-name`

3. **Install necessary dependencies**  
   `npm install`

4. **Update example.config.js and example.env**

5. **Run the application**  
   `npm run server-dev`
   `npm run server-dev`
